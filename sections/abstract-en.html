Abstract

The construction data gathering ontology is defined as the module that includes the concepts and relationships that relevant with the data gathering process in ther construction project. As designed in SSPO, the monitoring can be achieved by collecting the relevant construction data of instant event to find the real-time status of a domain object. The module is also a socket to link with other ICT ontologies in order to obtain data from ICT devices. Therefore, the SSN ontology is directly imported in this module.
